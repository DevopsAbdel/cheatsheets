# 35+ Cloud & DevOps Certifications: Role-Based Roadmap

This comprehensive guide details the key professional certifications required to navigate and scale your career across the top 8 specialized cloud and infrastructure engineering roles. The roadmap categories range from foundational skills to mid-level mastery and advanced architectural specializations.

---

### Foundation
* **Terraform Associate** (HashiCorp)
* **Kubernetes & Cloud Native Associate (KCNA)**

### Mid-level
* **Certified Kubernetes Administrator (CKA)**
* **Certified Kubernetes Application Developer (CKAD)**
* **AWS DevOps Engineer Professional**

### Advanced
* **Microsoft DevOps Engineer Expert**
* **NVIDIA AI Infrastructure**
* **GCP DevOps Engineer Professional**
